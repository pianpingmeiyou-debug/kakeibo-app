# リボン家計簿 仕様書 (v2.0)

## 1. アプリ概要
「リボン家計簿」は、日々の収支を楽しく記録し、AI（ロジック）による優しいアドバイスを受けることができるWebアプリです。

### 1.1 目的
日々の支出の「必要性」を可視化することで、ユーザーが自分に合った節約方法をポジティブに見つけられるようにする。

---

## 2. 実装済み機能

### 2.1 収支記録
- **支出/収入の切り替え**: ボタン一つで簡単に切り替え可能。
- **日付選択**: カレンダーから日付を選択。デフォルトは当日。
- **金額入力**: **100円単位で増減可能**（矢印キー対応）。
- **カテゴリ選択**: 食費、友達、日用品、美容、交通費、その他。
- **必要性評価**: 5段階の星（☆☆☆☆☆）で、その出費がどれだけ必要だったかを選択（支出のみ）。

### 2.2 ダッシュボード
- **今月の成果**: 支出目標に対する進捗をパステルカラーのバーで可視化。
- **収支サマリー**: 今月の総支出と総収入を大きく表示。
- **AIミニコメント**: 進捗に応じた一言アドバイス。
- **最近の記録**: 直近の取引をリスト表示。

### 2.3 分析・グラフ
- **カテゴリ別円グラフ**: Chart.jsを使用。ホバーで詳細金額を確認可能。
- **月次切り替え**: 前後の月のデータに簡単にアクセス。
- **AI詳細評価**: 月末にボタン一つで、その月の収支と必要性に基づいた詳細なアドバイスを生成。

### 2.4 カレンダー・目標設定
- **月間カレンダー**: 日ごとの収支状況を色分け（収入が多い日は青、支出が多い日は赤）して表示。
- **目標設定**: 収入目標、および支出の最小・最大目標を設定可能。

## 3. デザイン仕様
- **コンセプト**: かわいく、優しく、誰でも使いやすい。
- **配色**: 
  - さくら色 (#FFB7C5) - メインカラー
  - ミントグリーン (#B5EAD7)
  - パステルイエロー (#FDFD96)
  - ラベンダー (#E0BBE4)
- **UIコンポーネント**: 丸みを帯びたボタンとカード、グラスモルフィズム（半透明）のヘッダー・ナビゲーション。

---

## 4. 技術構成
- **Frontend**: HTML5, Vanilla JavaScript (ES6+), CSS (Tailwind Play CDN)
- **Icons**: Lucide Icons
- **Graphs**: Chart.js
- **Data**: Browser LocalStorage (サーバー不要、即座に動作)
